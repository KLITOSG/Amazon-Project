# Amazon Project

This repository contains a course-provided Amazon-style storefront starter project. The HTML, CSS, data files, backend JSON, and assets were supplied as ready-made code, and the learning goal is to add only the JavaScript needed to make the page interactive and fully functional.

## Project Context

- Built from course starter files for an Amazon-style shopping experience.
- The existing structure includes product pages, checkout, orders, tracking, styling, product data, and image assets.
- My task in the course is to write the JavaScript logic that brings the prepared interface to life.

## Technologies

- HTML5
- CSS3
- JavaScript
- JSON
